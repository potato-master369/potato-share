# win98 theme for Potato Share

## About

`win98` is a sample theme meant to showcase that Potato Share can be themed. To apply me, run the following in wherever you cloned the repo:
```bash
./applytheme.sh win98
```

## Legal

This theme was made using `98.css` from [UNPKG](https://unpkg.com). Some parts of the theme may have been created with the assistance of Artificial Intelligence (AI).

## Note

Please, PLEASE, do not apply malicious themes you have never seen before. They can do things like run JS code to delete everything you own. So don't.
